## 🆕 NEW PLUGINS ADDED

### 1. **Admin Plugin** (`admin.js`)
- `.afk <reason>` - Set away status
- `.back` - Return from AFK
- `.settings` - Bot configuration
- `.prefix` - Prefix management
- `.help` - Help center

### 2. **AI Plugin** (`ai.js`)
- `.ai <question>` - AI responses
- `.gpt <question>` - ChatGPT style

### 3. **Anime Plugin** (`anime.js`)

- `.manga <name>` - Search manga
- `.character <name>` - Character info
- `.waifu` - Random waifu
- `.husbando` - Random husbando

- `.calc <math>` - Calculator

- `.flip` / `.coin` - Coin flip
- `.dice` - Dice roll (1-6)

- `.guess` - Number guessing

### 8. **Leaderboard Plugin** (`leaderboard.js`)

### 10. **Moderation Plugin** (`moderation.js`)
- `.mute @user` - Silence members
- `.unmute @user` - Restore voice
- `.warn @user` - Issue warnings
- `.warnings` - Warning system
- `.block @user` - Block members

### 11. **Music Plugin** (`music.js`)
